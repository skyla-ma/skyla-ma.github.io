Scaling Monosemanticity — Technical Notes

The goal of the work is to address polysemanticity, where individual neurons respond to many unrelated patterns, which makes internal model representations difficult to interpret. The authors aim to separate these mixed signals into features that correspond more cleanly to specific concepts.

Method:

Sparse autoencoders (SAEs) are applied to activations from a layer of Claude 3. The autoencoder is trained to reconstruct the original activations under a sparsity constraint, meaning only a small number of learned features can be active at once. Each activation is represented as a combination of a few elements from a large learned dictionary.

Findings:

  • Scaled to a frontier model (Claude 3) and produced millions of learned features.
  • Features correspond to specific patterns such as programming topics, places, medical language, and behavioral signals.
  • Features activate selectively when related content appears in input.

Causal testing:

During inference, features can be forced to remain active (“clamped”), which changes model outputs in ways consistent with the feature’s associated concept.

Limitations:

  • Some features are not interpretable.
  • Some rarely activate (“dead features”).
  • Some still mix multiple concepts.
  • The process is computationally expensive.

Overall, sparse autoencoders can decompose internal activations into sparse features that correspond to identifiable patterns and can be measured and manipulated.
