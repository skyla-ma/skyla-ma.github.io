Scaling Monosemanticity — Technical Notes

The goal of the work is to address polysemanticity, where individual neurons respond to many unrelated patterns, which makes internal model representations difficult to interpret. The authors aim to separate these mixed signals into features that correspond more cleanly to specific concepts.

Method

They apply sparse autoencoders (SAEs) to activations from a layer of Claude 3. The autoencoder is trained to reconstruct the original activations, but under a sparsity constraint: only a small number of learned features can be active at a time. The model therefore represents each activation vector as a combination of a few elements from a large learned dictionary.

Because only a limited number of features are allowed to fire, the system is encouraged to represent signals using more specific and distinct components rather than blended ones. Each learned feature corresponds to a direction in activation space that contributes to reconstructing the original signal.

Findings

Scale

The method is applied to a large production model (Claude 3), producing millions of learned features. This extends earlier work that demonstrated similar behavior in smaller transformer models.

Granularity

Many learned features correspond to specific patterns in data, including:

  • programming concepts and error types
  • named entities and places
  • transportation systems
  • medical and scientific topics
  • social and behavioral patterns such as persuasion or deception

These features activate selectively when related content appears in the input.

Causal testing

To test whether features influence model behavior, the authors intervene during inference by forcing certain features to remain active (“clamping”). This changes the model’s outputs in ways consistent with the feature’s associated concept. The interventions show that these features participate in generating behavior, not just correlating with it.

Coverage and limitations

  • Not all learned features are interpretable.
  • Some features rarely activate (“dead features”).
  • Some still combine multiple concepts rather than representing a single one.
  • Training SAEs and running them across layers is computationally expensive.

Overall technical result

Sparse autoencoders can decompose internal activations of a frontier language model into a large set of sparse features that correspond to identifiable patterns in data. These features can be measured, analyzed, and causally manipulated, providing a method for studying internal representations at scale.
